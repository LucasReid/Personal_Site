Database and Application Development

My overall part in this project was to style and format the information on the page as well as designing the layout for the website

I designed the home page layout and information boxes as well as the menu feature. When hovered over the Fraternity and sorority boxes change image and when clicked on drop down information specifically pertaining to the corresponding organization. Beneath the title the users name will appear if they are logged in. The menu is an off canvas menu that pushes the page to the right when opened and the items in the menu change depending on whether or not a user is logged in.

On the Log-In page I created the drop down boxes, input boxes, buttons, and pop out modal. The website has two more pages which are the user and Admin pages. These pages correspond to the user logged in; the user page shows information pertaining to that specific user as well as a calendar that is linked to Google. I retrieved data pertaining to the specific user from the database the team created and displayed that on screen, to create a type of profile. I created a similar layout for the Admin page while giving them the ability to modify information in the database for their specific Fraternity or Sorority users.
